# Instagram Ad Image Design Prompt

## Quick Copy-Paste Prompt for AI Image Generators

```
Create a professional Instagram ad image (1080x1080px square format) for a web development agency targeting small business owners in Casablanca, Morocco.

VISUAL ELEMENTS:
- Clean, modern, mobile-first design
- Smartphone mockup showing a fast-loading app/website interface
- Prominent WhatsApp icon in green (#25D366) with "Commandes WhatsApp" text
- Lightning bolt or speed indicator showing "10x plus rapide"
- Price display: "À partir de 9,000 MAD" in bold, readable font
- "Basé à Casablanca" badge or text for local trust
- Professional color scheme: Deep blue (#1e40af), WhatsApp green (#25D366), white background
- Minimal text overlay (Instagram-friendly, not too much text)

LAYOUT:
- Center: Smartphone mockup with app interface visible
- Top left/right: Headline "Site Ultra-Rapide + WhatsApp"
- Bottom: Key benefit callout with WhatsApp icon
- Clean, uncluttered design with plenty of white space

STYLE:
- Modern, tech-forward but approachable
- Professional yet friendly (appeals to traditional business owners)
- High contrast for mobile feed visibility
- Instagram-native aesthetic (bright, engaging, scroll-stopping)

TEXT OVERLAY (minimal):
- Main headline: "Site Ultra-Rapide + Commandes WhatsApp"
- Subheadline: "À partir de 9,000 MAD"
- Trust signal: "Basé à Casablanca"

AVOID:
- Too much text (Instagram penalizes text-heavy images)
- Cluttered design
- Generic stock photos
- Hard-to-read fonts
- Dark backgrounds (less visible in feed)
```

---

## Detailed Design Specifications

### Dimensions:
- **Format**: Square (1080x1080px) - Instagram Feed
- **Alternative**: Story format (1080x1920px) if needed

### Color Palette:
- **Primary Blue**: #1e40af (Deep blue for tech/professional)
- **WhatsApp Green**: #25D366 (Prominent, recognizable)
- **White**: #FFFFFF (Clean background)
- **Accent**: #fbbf24 (Gold for price/urgency if needed)

### Typography:
- **Headline**: Bold, sans-serif (Montserrat, Inter, or similar)
- **Body**: Clean, readable (16-20px minimum)
- **Price**: Extra bold, large (24-32px)

### Key Visual Elements:

1. **Smartphone Mockup** (Center):
   - Modern iPhone or Android device
   - App interface showing:
     - Fast loading indicator
     - WhatsApp button/icon visible
     - Clean, modern UI
     - Product/menu items (if restaurant/café theme)

2. **WhatsApp Integration** (Prominent):
   - Green WhatsApp icon (#25D366)
   - "Commandes WhatsApp intégrées" text
   - Arrow or connection line to phone

3. **Speed Indicator**:
   - Lightning bolt icon
   - "10x plus rapide" text
   - Visual speed lines or motion blur effect

4. **Price Display**:
   - "À partir de 9,000 MAD" in bold
   - Currency symbol or "MAD" clearly visible
   - Not too small (readable on mobile)

5. **Trust Elements**:
   - "Basé à Casablanca" badge
   - Or "Local" indicator
   - Subtle but visible

### Layout Structure:

```

│                         │

│  [Speed Indicator]      │
│                         │

```
