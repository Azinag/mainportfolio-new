# Logo Generation Prompt for Azinag Web Solutions

## AI Image Generator Prompt

Use this prompt with DALL-E, Midjourney, Stable Diffusion, or similar AI image generators:

```
Create a professional, minimalist logo for "Azinag Web Solutions" - a web development agency specializing in Progressive Web Apps (PWAs), custom mobile apps, and fast websites. 

Design requirements:
- Simple, geometric icon or symbol that works at 16x16 pixels (favicon size)
- Clean, modern, tech-forward aesthetic
- Single-color or two-color design (works in light and dark modes)
- Scalable vector-style design
- No fine details or thin lines that disappear at small sizes
- Professional and trustworthy appearance
- Represents: web development, technology, innovation, speed, modern apps

Style options:
1. Abstract geometric shape (triangle, hexagon, or custom shape) with "A" letterform integration
2. Minimalist icon combining web/tech elements (grid, code brackets, or app icon style)
3. Stylized "A" letterform with tech/web elements subtly integrated
4. Modern badge or shield shape with clean typography

Color palette suggestions:
- Primary: Deep blue (#1e40af) or vibrant purple (#7c3aed) for tech/professional feel
- Alternative: Gradient from blue to purple
- Must work in monochrome (black/white) for dark mode compatibility

Technical specifications:
- Square format (1:1 aspect ratio)
- High contrast for visibility at small sizes
- No gradients or shadows that blur at favicon size
- Bold, clear shapes
- Recognizable silhouette even when scaled down to 16x16px

Reference style: Similar to Vercel's triangle logo, Stripe's minimalist design, or modern SaaS company logos - clean, memorable, professional.
```

## Additional Notes for Logo Creation

### Favicon Requirements:
- **Size**: 16x16, 32x32, 48x48, 180x180 pixels (multiple sizes needed)
- **Format**: PNG with transparency, or ICO format
- **Colors**: Should work in both light and dark browser themes
- **Simplicity**: Must be readable at 16x16px - avoid text, use symbols only

### Google Search Results:
- Logo appears in Google Business Profile and search results
- Needs to be recognizable at small sizes
- Should represent your brand clearly
- Professional appearance builds trust

### Implementation Tips:
1. Generate the logo in high resolution (1024x1024 or higher)
2. Create multiple versions:
   - Full color version
   - Monochrome (black) version for dark mode
   - Simplified version for favicon
3. Test at 16x16 pixels to ensure readability
4. Consider creating a wordmark version (logo + text) for header use

### Brand Identity Elements to Consider:
- **Azinag** = Web Solutions, PWAs, Mobile Apps
- **Morocco-based** but serves international clients
- **Modern tech stack**: Next.js, React, Supabase
- **Key differentiator**: Fast, no WordPress, WhatsApp integration
- **Values**: Speed, quality, innovation, reliability
